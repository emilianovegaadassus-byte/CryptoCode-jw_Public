# Contribuciones

¡Gracias por tu interés en mejorar CryptoCode!

---

## 🧭 ¿Cómo contribuir?

El **código fuente de la plataforma no es público**, pero valoramos enormemente tu retroalimentación. Puedes contribuir de las siguientes maneras:

### 1. Reportar bugs o sugerir mejoras
- Abre un **issue** en este repositorio usando la plantilla correspondiente.
- Describe claramente el problema o la idea, y si es posible, pasos para reproducirlo.

### 2. Mejorar la documentación
- Si encuentras errores o falta de claridad en nuestros archivos, puedes abrir un issue o un pull request con correcciones.

### 3. Participar en discusiones
- Usa la pestaña **Discussions** (si está habilitada) para compartir ideas, casos de uso o feedback.

---

## 📝 Proceso para issues

1. Busca si ya existe un issue similar.
2. Si no, crea uno nuevo usando la plantilla.
3. Sé específico: incluye logs, capturas de pantalla si es relevante, y la versión del sistema.

---

## ✅ Lo que no se acepta

- Publicación de vulnerabilidades sin seguir [SECURITY.md](SECURITY.md).
- Spam o autopromoción.
- Código no solicitado (ya que el código principal es privado).

---

**Gracias por ayudarnos a construir una herramienta mejor.**
