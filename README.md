# CryptoCode — AI-Powered Crypto Trading Platform

[![Website](https://img.shields.io/badge/Web-cryptocode--jw.com-blue)](https://cryptocode-jw.com)
[![Status](https://img.shields.io/badge/Production_Ready-v2.5.0-brightgreen)]()
[![License](https://img.shields.io/badge/License-MIT-yellow.svg)](LICENSE)

CryptoCode es una plataforma profesional de trading automatizado de criptomonedas que combina **análisis técnico**, **machine learning** y **gestión de riesgo institucional** para generar recomendaciones claras y accionables. Todo sin tocar tus fondos.

---

## 🚀 ¿Qué hace CryptoCode?

- Escanea **~600 pares** cada 15 minutos con indicadores técnicos avanzados (RSI, MACD, Supertrend, PMAX, volatilidad).
- Filtra señales con modelos de **Machine Learning** (Signal Classifier, LSTM, Market Regime) para reducir ruido.
- Aplica **gestión de riesgo profesional** con stop-loss, take-profit, trailing stops, rebalanceo dinámico y protección de heat.
- Permite ejecutar **múltiples estrategias** simultáneas (hasta 5 en plan Pro, ilimitadas en Premium) con asignación de capital inteligente.
- Ofrece un **dashboard completo** con métricas en tiempo real, equity curves, posiciones activas y auditoría total.
- **Nunca toca tus fondos** – las recomendaciones son manuales o vía testnet (y opcionalmente live trading solo para usuarios Premium con confirmación explícita).

## ✨ Características destacadas

| Área | Funcionalidad |
|------|---------------|
| **Análisis técnico** | 6 indicadores principales, cálculo en 15min, 500+ activos. |
| **Machine Learning** | 3 modelos entrenados con auto-retrain semanal, accuracy 57-75%. |
| **Risk Management** | 15 funciones integradas (SL/TP, trailing, heat, partial exits, etc.). |
| **Multi‑estrategia** | Coordinación automática de hasta 5 estrategias con heat allocation. |
| **Seguridad** | API keys encriptadas AES-256-GCM, JWT httpOnly, auditoría completa. |
| **Dashboard** | 11 páginas, diseño unificado, gráficos interactivos, modo testnet. |

## 🧪 Quick start

1. **Prueba la versión gratuita** en [cryptocode-jw.com](https://cryptocode-jw.com).
2. Crea una cuenta y elige una estrategia predefinida (Aggressive, Conservative, etc.).
3. Conecta tu cuenta de **Binance Testnet** (opcional) para simular órdenes reales.
4. Recibe recomendaciones en el dashboard y decide si ejecutarlas.
